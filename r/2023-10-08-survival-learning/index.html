泰坦尼克号的生存数据

让我们以泰坦尼克号数据集分析问题的分类模型中的DALEX包为例，引用的数据集是来自其中的titanic。这份数据是从stablelearner包复制而来。

运行以上代码，Rstudio的帮助界面中会出现titanic_imputed的详细信息，我们可以据此了解该数据集中各变量的含义。

各变量含义
  1. gender：性别
  2. age：年龄
  3. class：乘客舱位等级或船员服务类型
  4. embarked：登船地点
  5. fare：票价（船员、音乐家和船厂员工的为0）
  6. sibsp：同胞或配偶的数量
  7. parch：父母或子女的数量
  8. survival：是否生存

泰坦尼克号的生存模型

使用随机森林模型试试。

Code 
library(ranger)
model_titanic_rf <- ranger(survived ~ gender + age + class + embarked +fare + sibsp + parch,  data = titanic_imputed, classification = TRUE)

变量效应

由上图，我们知道性别是最突出的特征，其他三个比较突出的特征分别是舱位等级、年龄和票价。试着探究模型响应变量和这些特征之间分别的联系。

这种单变量关系可以用variable_effect()来计算。

实例层面的解释

看看从C处登船的头等舱的8岁男性的模型预测细分解释。

Code 
new_passanger <- data.frame(
  class = factor("1st", levels = c("1st", "2nd", "3rd", "deck crew", "engineering crew", "restaurant staff", "victualling crew")),
  gender = factor("male", levels = c("female", "male")),
  age = 8,
  sibsp = 0,
  parch = 0,
  fare = 72,
  embarked = factor("Southampton", levels = c("Belfast", "Cherbourg", "Queenstown", "Southampton"))
)

sp_rf <- predict_parts(explain_titanic_rf, new_passanger)
plot(sp_rf)

由上图可以看出，这位乘客最重要的特征是年龄和性别。毕竟他的生存几率比一般乘客要高，主要是因为年龄小，虽然是男性。
